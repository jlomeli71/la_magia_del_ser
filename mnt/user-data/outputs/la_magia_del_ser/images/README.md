# Imágenes del Álbum — El Estado del Ser

Esta carpeta contiene los prompts para generar las imágenes del álbum usando herramientas de IA como Midjourney, DALL-E o Adobe Firefly.

## Estilo Visual General del Álbum

Todas las imágenes deben seguir esta estética coherente:
- **Paleta:** Tonos suaves, espirituales, etéreos (azules, dorados, verdes suaves, blancos)
- **Estilo:** Arte digital espiritual, acuarela digital, naturaleza sagrada
- **Atmósfera:** Paz, luz, expansión, naturaleza hawaiana y universal
- **Formato:** Horizontal (16:9 o 3:2) para el booklet del álbum
- **Resolución:** Mínimo 1920x1080 px

## Herramientas Recomendadas para Generar las Imágenes

1. **Midjourney** — Mejor calidad artística. Copiar el prompt directamente en Discord
2. **DALL-E 3** (via ChatGPT) — Fácil de usar, buen resultado
3. **Adobe Firefly** — Excelente para imágenes sin copyright
4. **Stable Diffusion** — Gratuito, requiere instalación local
